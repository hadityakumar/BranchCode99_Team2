# Transportion-service

Transportion-service is a web application developed using React.js for managing Transportion timings.

Currently, two official plugins are available:
## Installation

1. **Run react server:**

    ```bash
    cd Frontend
    npm install
    npm run dev


2. **Run Backend server:**

    ```bash
    cd Backend
    npm install
    nodemon server.js
